# shooter
Alien Shooter is a simple action-packed shooting game where players fight waves of alien invaders. Use powerful weapons, dodge enemy attacks, and earn points by defeating aliens. Progress through increasingly challenging levels, unlock upgrades, and protect Earth from the alien threat in this exciting arcade adventure.
